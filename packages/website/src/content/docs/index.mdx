## Features

<CardGrid stagger>
  <Card title="Hooks" icon="setting">
    Auto-lint on file changes, quality review on session end, version tracking
  </Card>
  <Card title="Skills" icon="puzzle">
    Structured workflows for TDD, debugging, refactoring, brainstorming
  </Card>
  <Card title="Commands" icon="rocket">
    Slash commands for linting, auditing, drift detection
  </Card>
  <Card title="Multi-language" icon="translate">
    JavaScript/TypeScript, Python (Ruff), Go (golangci-lint)
  </Card>

## Quick Install

```bash
bunx safeword@latest setup
```

Your project will have hooks, skills, guides, and linting configured in seconds.
