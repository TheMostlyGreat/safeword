Quick actions available via slash syntax in Claude Code.

| Command            | Purpose                                 |
| ------------------ | --------------------------------------- |
| `/lint`            | Run linters and formatters on your code |
| `/audit`           | Architecture and dead code analysis     |
| `/drift`           | Check if docs match the actual codebase |
| `/quality-review`  | Deep code review with web research      |
| `/cleanup-zombies` | Kill zombie dev server processes        |

## /lint

Routes files to the correct linter based on extension:

- **TypeScript/JavaScript** → ESLint + Prettier
- **Python** → Ruff + mypy
- **Go** → golangci-lint

## /audit

Analyzes architecture compliance, dead code, and dependency issues.

## /drift

Compares `ARCHITECTURE.md` against actual structure. Creates or updates docs if missing.

## /quality-review

Checks library versions, security patterns, and best practices against official docs.

## /cleanup-zombies

Kills orphaned dev servers, stuck test runners, and port conflicts.
