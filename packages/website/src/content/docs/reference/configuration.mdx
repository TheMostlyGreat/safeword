## File Categories

### Owned Files

Overwritten on every `safeword upgrade`:

- `.safeword/hooks/*`, `.safeword/guides/*`, `.safeword/templates/*`
- `.claude/skills/*`, `.claude/commands/*`
- `.cursor/rules/*`

### Managed Files

Created if missing, never overwritten:

- `eslint.config.mjs`, `.prettierrc`
- `.claude/settings.json`
- `AGENTS.md`

## Language Packs

Auto-detected from your project. Tracked in `.safeword/config.json`.

| Language   | Linting           | Type Checking |
| ---------- | ----------------- | ------------- |
| TypeScript | ESLint + Prettier | tsc           |
| Python     | Ruff              | mypy          |
| Go         | golangci-lint     | go vet        |

## ESLint Plugin

```js
import safeword from 'eslint-plugin-safeword';

export default [
  ...safeword.configs.recommendedTypeScript,
  ...safeword.configs.react, // optional
  ...safeword.configs.vitest, // optional
];
```

Available: `recommendedTypeScript`, `react`, `vitest`, `playwright`, `astro`

## Hook Configuration

Hooks are configured in `.claude/settings.json` under the `hooks` key. See [Hooks reference](/reference/hooks) for available hooks.

## Project-Specific Content

Content that won't be touched by upgrades goes in `.safeword-project/`.
